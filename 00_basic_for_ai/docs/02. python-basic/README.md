# Python Basic

AI 학습과 데이터 처리를 위한 Python 기초 실습 트랙입니다. 별도 로컬 환경을 구성하지 않고 Google Colab에서 notebook을 열어 코드를 실행할 수 있습니다.

## Colab Notebook

[Google Colab에서 Python 기초 실습 열기](https://colab.research.google.com/drive/1_XAfNegKjQqMy-Ra1SU6FjohX7UHWcPq?usp=drive_link)

## Learning Goal

- Python 코드의 기본 실행 흐름을 익힌다.
- notebook의 code cell과 output을 읽고 직접 실행한다.
- AI·데이터 학습에서 사용하는 Python 실습 환경에 익숙해진다.
- 실습 결과와 오류를 학습 기록으로 남긴다.

## How to Use

1. 위 Colab 링크를 연다.
2. Google 계정으로 로그인한다.
3. 원본을 보존하려면 `파일 > Drive에 사본 저장`을 선택한다.
4. 위에서 아래 순서로 code cell을 실행한다.
5. 실행 결과와 오류 메시지를 함께 확인한다.

> Notebook의 공개 범위는 Google Drive 공유 설정에 따라 달라집니다. 다른 사람이 열어야 한다면 링크 공유 권한을 별도로 확인해야 합니다.
